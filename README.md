\# 📒 Address Book Management System (C Project)



\## 📌 Overview

This is a simple Address Book Management System written in C.  

It allows users to store, search, edit, and delete contacts using file handling.



\---



\## 🚀 Features

\- Add new contact  

\- View all contacts  

\- Search contact  

\- Edit contact  

\- Delete contact  

\- File-based storage  



\---



\## 🛠️ Technologies Used

\- C Programming  

\- File Handling  

\- Structures  

\- Modular Programming  



\---



\## ⚙️ How to Run



\### Compile

```bash

gcc main.c contact.c file.c -o addressbook

\---

\###Run

```bash

./addressbook
