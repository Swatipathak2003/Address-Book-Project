# 📒 Address Book Management System (C Project)

## 📌 Overview
This project is a command-line based Address Book Management System developed in C.  
It allows users to efficiently manage contact details such as name, phone number, and email using file handling for persistent storage.

## 🚀 Features
- ➕ Add new contact  
- 📄 View all contacts  
- 🔍 Search contact by name  
- ✏️ Edit existing contact  
- ❌ Delete contact  
- 💾 Persistent storage using file handling (`contacts.txt`)  

## 🛠️ Technologies Used
- C Programming  
- File Handling  
- Structures  
- Modular Programming (.c and .h files)  

## 📂 Project Structure

├── main.c
├── contact.c
├── contact.h
├── file.c
├── file.h
├── contacts.txt


## ⚙️ How to Run

### Step 1: Compile
gcc main.c contact.c file.c -o addressbook

### Step 2: Run
./addressbook
